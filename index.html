Robotgame.net : Taandav

Taandav is the project to experiment with swarm robotics and python.

Starting contribution

Fork or Clone the repo and work, open issues, fix them, contribute in anyway you can.

$ cd folder_name
$ git clone https://github.com/kunaltyagi/robotgame.git
$ cd robotgame

Update Guidelines

Fetch before working. Other authors might have resolved something. And yeah, always push your code onto an unstable branch. Only after the code has been tested it should be pushed to the devel stable branch.
